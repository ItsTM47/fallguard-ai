# Backend

- api/: relay API
- api/config/: env/config
- api/middleware/: auth/error/upload middleware
- api/routes/: health/images/webhook routes
- api/services/: line/mlflow/image services
- api/utils/: shared HTTP utils
- database/: postgres schema/migration/event store
- services/: backend service modules
- models/: backend domain models
- uploads/: runtime images
